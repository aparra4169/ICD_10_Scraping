Note:

This code is provided primarily for morbidity coding. For use of this category reference should be made to the morbidity coding rules and guidelines in Volume 2.

Incl.:

  • Oophoritis
  • Parametritis
  • Pelvic peritonitis
  • Salpingitis
  • Salpingo-oophoritis
  • Sepsis
